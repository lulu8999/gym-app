# 微雪桌面终端项目 — 进度记录

## 状态：待采购（全家桶已确认，外壳已设计）

## 项目演进

- **V1 定位**（2026-06-13）：桌面小终端，显示天气/课表/消息
- **V2 定位**（2026-06-14）：**全能法治助手体系——物理接口层**

Lulu 希望将 ESP32 纳入完整的法治助手生态（VPS + Mac + 手机 + Win本 + ESP32），
ESP32 的角色从"桌面小玩具"升级为"物理世界接口"。

## 设备矩阵

```
VPS（云端大脑）  ← Hermes AI / 数据库 / Web服务 / 长期存储
Mac（本地存储）   ← 备份 / 解密分析 / 文书生成 / 类案检索
手机（随身）      ← 拍照取证 / 录音录像 / 移动办公
Win本（开发）     ← 开发调试 / 备用终端
ESP32（桌面常驻） ← 传感器采集 + 屏幕显示 + 语音交互 + 物理控制
```

ESP32 不做重活，只做**采集、显示、交互**。计算和存储全交给 VPS/Mac。

## 外壳设计（OpenSCAD V2）
- 文件：`/root/esp32_case/terminal_case_v2.scad`
- 外部尺寸：约 83 × 60 × 28mm
- 设计原则：安装便利第一 + 后续扩展余量

### 外壳功能清单
- ✅ 电池卡槽（四壁+底托+拉手槽，推入即卡住）
- ✅ 走线槽（4mm宽，电池仓→板子仓→BAT接口）
- ✅ 螺丝柱（4×M2.5，间距60×41mm）
- ✅ 卡扣后盖（四边各3个卡扣，按入即固定）
- ✅ RESET/BOOT/BAT PWR 按钮孔（侧面，针戳）
- ✅ USB-C 开口 / SD 卡槽开口
- ✅ 喇叭格栅（背面）/ 通风孔阵列（后盖）
- ✅ 防滑脚垫槽（底部）/ 两侧扩展槽

### 安装顺序
1. 电池推入卡槽（线从走线槽引出）
2. 板子对准螺丝柱，拧 M2.5 螺丝
3. 电池线插 BAT 接口
4. 卡扣后盖按入
