# ESP32 外壳设计说明（V7 最终版 2026-06-14）

## V7 核心变更
- **外壳加宽**：从 100mm → **120mm**，容纳 EC11 独立区域
- **EC11 位置修正**：从主板正上方 → **右侧独立区域（x=76~106）**
- **原因**：EC11 叠在主板上方会遮挡 2.8 寸触摸屏
- **灯环余量**：φ49.9mm 灯环右边缘距外壳壁 5.1mm

## 设计文件位置
- 2D 布局图：`/root/esp32-shell/assembly_3d_v7.png`
- OpenSCAD 源码（旧V6）：`/root/esp32-shell/esp32_case.scad`
- Windows 桌面：`esp32_case.scad`、`esp32_front_case.stl`、`esp32_back_cover.stl`

## STL 导出工作流（VPS → Windows）

```bash
# 1. 传输 .scad 文件到 Windows 桌面
scp -P 2222 /path/to/file.scad user@win:"C:/Users/陆海天/Desktop/"

# 2. 导出（OpenSCAD 完整路径）
ssh -p 2222 user@win '"C:\Program Files\OpenSCAD\openscad.com" -o "C:\Users\陆海天\Desktop\front.stl" "C:\Users\陆海天\Desktop\export_front.scad"'
```

⚠️ VPS 无 OpenSCAD，必须通过 SSH 在 Windows 上导出。

## 10 组内部固定结构

| 编号 | 结构 | 类型 | 尺寸/说明 |
|------|------|------|-----------|
| ❶ | 电池仓挡板 | 四面围栏 | 高12mm，底部限位 |
| ❷ | ESP32 安装柱 ×4 | M2.5 铜柱 | 高18mm，孔距60×41mm |
| ❸ | EC11 安装柱 ×4 | M3 铜柱 | 高22mm，独立于主板 |
| ❹ | 灯环卡槽 | 环形凹槽 | φ49.9mm，顶面嵌入 |
| ❺ | DHT22 卡槽 | L型卡条 | 右壁，上下卡条+底托 |
| ❻ | BME680 卡槽 | L型卡条 | 右壁下方，独立卡槽 |
| ❼ | 前面传感器托盘 | 托盘式 | TSL25911 + VL53L0X |
| ❽ | INMP441 卡槽 | L型卡槽 | 左壁，拾音孔朝外 |
| ❾ | 振动马达卡槽 | L型卡槽 | 右下角 |
| ❿ | 走线槽 | 主槽+分支 | 底部纵横走线 |

## 结构设计

| 部位 | 设计 |
|------|------|
| 前盖 | 屏幕开口+旋钮开口+灯环透明窗口 |
| 后盖 | 卡扣式，不用螺丝 |
| EC11 独立 | 自有4个安装柱，与主板完全分离 |
