# 外接模块接线表（2026-06-14 最终版）

## INMP441 麦克风（I2S）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| SCK | GPIO15 | I2S 时钟 |
| WS | GPIO18 | I2S 字选择 |
| SD | GPIO44 (RXD) | I2S 数据 |
| L/R | GND | 选左声道 |

## DHT22 温湿度（单总线）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| DATA | GPIO15 | 单总线数据 |

⚠️ GPIO15 和 INMP441 SCK 冲突，实际只能选一个用。建议 DHT22 数据改到其他 GPIO，或用 BME680 替代（I2C 接口，不占 GPIO15）。

## TSL25911 光照（I2C）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| SDA | GPIO11 | I2C 数据 |
| SCL | GPIO10 | I2C 时钟 |
| INT | 不接 | 中断引脚（可选） |

## BME680 四合一（I2C）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| SDA | GPIO11 | I2C 数据（和 TSL2591 共用） |
| SCL | GPIO10 | I2C 时钟（和 TSL2591 共用） |
| ADDR | 不接 | 默认地址 0x76 |

## VL53L0X 测距（I2C）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| SDA | GPIO11 | I2C 数据（共用总线） |
| SCL | GPIO10 | I2C 时钟（共用总线） |

⚠️ 默认地址 0x29，可能和 TSL2591 冲突。需要通过 XSHUT 引脚改地址或代码中指定。

## EC11+OLED 旋钮（GPIO + I2C）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| SDA | GPIO11 | OLED 数据（I2C） |
| SCL | GPIO10 | OLED 时钟（I2C） |
| PSH | GPIO20 | EC11 按键 |
| TRA | GPIO15 | EC11 A相 |
| TRB | GPIO18 | EC11 B相 |
| BAK | GPIOx | 返回键（可自定义） |
| CON | GPIOx | 确认键（可自定义） |

## WS2812B 16灯环（单总线）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 5V (VBUS) | 供电（必须5V） |
| GND | GND | 地线 |
| DI | GPIO15 | 数据输入 |

## 振动马达（GPIO）

| 模块引脚 | ESP32 引脚 | 说明 |
|---------|-----------|------|
| VCC | 3V3 | 供电 |
| GND | GND | 地线 |
| IN | GPIO18 | 高电平触发 |

## 引脚冲突汇总

| GPIO | 冲突模块 | 解决方案 |
|------|---------|---------|
| GPIO15 | INMP441 SCK / DHT22 DATA / EC11 CLK / WS2812B DI | 分时复用或换引脚 |
| GPIO18 | INMP441 WS / EC11 DT / 振动马达 IN | 分时复用或换引脚 |

**建议**：
1. DHT22 可以用 BME680 替代（I2C 接口，不占 GPIO15）
2. INMP441 和 WS2812B 不会同时使用（录音时关灯）
3. EC11 和 INMP441 可以分时复用（录音时旋钮暂停）
