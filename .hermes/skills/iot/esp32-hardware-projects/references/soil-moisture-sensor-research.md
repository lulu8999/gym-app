## 推荐方案

### 方案一：有线方案（最便宜，花盆集中时推荐）
- 主控：ESP32-C3 SuperMini（¥10）
- 传感器：电容式土壤湿度传感器 ×N（¥3-5/个）
- 连接：杜邦线公对母
- 每盆成本：¥3-5
- 优点：稳定、简单、便宜
- 缺点：需要走线，花盆不能太分散

### 方案二：DIY 无线节点（花盆分散时推荐）
- 每盆一个节点：ESP32-C3 SuperMini（¥10）+ 电容传感器（¥3-5）+ CR2032 电池座（¥1）
- 主控：ESP32 接收蓝牙广播数据
- 每盆成本：¥14-16
- 优点：无线，灵活
- 缺点：需要写 BLE 广播代码，电池需定期换

### 方案三：米家花花草草（不推荐）
- 太贵（¥90/个），功能过剩（光照/肥力对花盆监测非必需）

## 淘宝搜索关键词
- 传感器：`电容式 土壤湿度传感器` 或 `电容式 土壤湿度检测`
- 主控：`ESP32-C3 SuperMini`
- 电池座：`CR2032 电池座 2P`
- 杜邦线：`杜邦线 公对母 20cm`

## 技术要点
- 电容式比电阻式好：不会腐蚀，寿命长
- ESP32-C3 支持 BLE 5.0，可做蓝牙网关
- 传感器输出模拟量(ADC)，ESP32-C3 有 ADC 引脚可直接读
- 阈值建议：土壤湿度 < 30% 时发预警

## 通知方案
- 个人微信无官方 API
- 推荐用**企业微信 webhook** 推送，效果等同微信消息
- 替代：Server酱、PushPlus、Bark 等推送服务
