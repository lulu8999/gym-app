# 医学考研（西医综合306）生理学资料整理与预测

## 🎯 触发条件

当用户提出以下需求时激活本技能：
- "整理生理考研资料"
- "306生理学真题分析"
- "预测考研重点"
- "复习资料整理"
- 提到天天师兄/贺银成/医考帮等考研辅导资料

## ⚠️ 关键工作流偏好

**【必须遵守】**用户明确要求**增量式报告**：
- 每完成一个阶段，立即报告进度
- 禁止等全部完成才一次汇报
- 格式："第X步完成！完成了...，正在做...下一步..."

## 📋 执行流程

### 第一步：确认需求
必须明确以下信息：
- [ ] 考试类型：西医综合（306）vs 自主命题
- [ ] 目标年份：2027考研（目前大多数）
- [ ] 教材版本：人卫第10版（主流）
- [ ] 输出格式：Markdown / Word / Excel

**报告：**“第一步完成！已确认考试类型为XXX，目标年份XXX...”

### 第二步：收集现有资料
1. 检查用户目录是否已有相关资料
2. 如有PDF讲义，提取文字内容
3. 整理已有笔记/思维导图/真题

**工具命令：**
```bash
# PDF提取
pdftotext 输入.pdf 输出.txt

# 创建工作目录
mkdir -p /root/users-data/[username]/生理考研资料/
```

**报告：**“第二步完成！已提取PDF内容XXX行，发现第X章至第Y章内容...”

### 第三步：分析历年真题规律
基于公开回忆版真题，统计：
- 各12章节历年分值分布
- 年均出题频次
- 难度评估

**章节分值参考（2006-2025）：**
| 章节 | 年均分值 | 频次 | 难度 |
|------|---------|------|------|
| 循环 | ~8分 | ⭐⭐⭐⭐⭐ | 高 |
| 神经 | ~7分 | ⭐⭐⭐⭐⭐ | 高 |
| 内分泌 | ~6分 | ⭐⭐⭐⭐ | 中 |
| 细胞 | ~6分 | ⭐⭐⭐⭐ | 高 |
| 呼吸 | ~5分 | ⭐⭐⭐ | 中 |
| 泌尿 | ~4分 | ⭐⭐⭐ | 中 |

**报告：**“第三步完成！已统计近20年分值分布，最高频章节为XXX...”

### 第四步：预测重点考点
基于以下维度预测当年重点：
1. **近年趋势**：连续出现的高频考点
2. **周期性规律**：偶年/间年出现的考点
3. **教材变化**：第10版新增/调整内容

**第一梯队（必考）：**
- Starling机制 + 心肌前负荷调节
- EPSP与IPSP对比
- 特异性vs非特异性投射系统
- 甲状腺激素调节
- 氧解离曲线右移

**报告：**“第四步完成！预测第一梯队考点X个，第二梯队X个...”

### 第五步：整合生成文档
将以下内容整合为一份完整资料：
1. 历年分值分布表
2. 2027重点预测
3. 各12章核心考点（含口诀）
4. 易错点汇总
5. 答题技巧
6. 复习规划

**报告：**“第五步完成！已生成整合版资料，共XXX行，位于XXX路径...”

### 第六步：发送确认
1. 发送文件给用户查看
2. 等待用户确认无误

**报告：**“文件已发送！请查看确认，无误后将清理临时文件。”

### 第七步：清理临时文件
**【必须等待用户确认】**
用户确认后再执行：
```bash
rm /tmp/生理_提取.txt  # PDF提取的临时文件
```

**报告：**“临时文件已清理，任务完成！”

## ⚠️ 常见陷阱

1. **忽略增量报告偏好**：用户明确要求"做一步报告一步"，禁止一次性完成
2. **擅自清理文件**：必须等待用户确认无误后再清理
3. **混淆考试类型**：确认是西医综合306还是自主命题，两者考点不同
4. **口诀错误引用**：确认教材版本（第9版vs第10版口诀可能不同）
