# 中国法律网站爬取能力测试（2026-06-15）

测试工具：Crawl4AI 0.8.9 + curl + Jina Reader API

## 测试结果汇总

| 网站 | URL | 状态 | 可爬内容 | 限制 |
|------|-----|------|---------|------|
| 中国法院网 | chinacourt.org | ✅ 可用 | 法院新闻、审判资讯 (30K+ chars) | 个别文章WAF返回403 |
| 中国司法大数据服务网 | data.court.gov.cn | ✅ 可用 | 研究报告标题和列表 | PDF需注册下载 |
| 人民法院案例库 | rmfyalk.court.gov.cn | ⚠️ 部分 | 首页结构(3927篇案例分类) | 搜索和详情页JS动态加载 |
| 国家法律法规库 | flk.npc.gov.cn | ⚠️ 部分 | 框架页面(法律310/行政法规610/地方法规15755) | 法条正文全JS加载，抓不到 |
| 裁判文书网 | wenshu.court.gov.cn | ❌ 不可用 | 仅登录框和导航 | 强制登录+滑块验证+反爬风控极强 |
| 中国法律服务网 | 12348.gov.cn | ❌ 连接失败 | 无 | ERR_CONNECTION_CLOSED |

## 关键发现

### 裁判文书网 (wenshu.court.gov.cn)
- 反爬最强的法律网站，Python Selenium自动化也需处理登录+滑块
- 2024年后访问限制更严，需要手机号注册
- **建议放弃爬取**，改用人民法院案例库替代

### 国家法律法规库 (flk.npc.gov.cn)
- 前后端分离架构，搜索结果通过JS API动态渲染
- Crawl4AI抓到的只有框架HTML（"很抱歉，未查询到任何数据"）
- **可行方案**：用CamoFox模拟浏览器执行JS后抓取
- 或者用Jina Reader API（但该站返回内容也有限）

### 人民法院案例库 (rmfyalk.court.gov.cn)
- 2024年新建的公开案例库，收录3927篇案例
- 分类：刑事/民事/行政/国家赔偿/执行
- 搜索和详情页是JS渲染，需要浏览器自动化
- **可行方案**：CamoFox逐个爬取，量不大可以接受

### 中国法院网 (chinacourt.org)
- **最容易爬**，公开新闻资讯无需登录
- 地方法院报道、审判动态、法学文章都有
- 个别页面有WAF防护(403)，大部分可正常抓取

## 实际可行方案（法制助手项目）

1. **法律资讯** → 直接爬中国法院网（公开，无需登录）
2. **案例检索** → CamoFox模拟浏览器爬人民法院案例库（3927个，可控）
3. **法规条文** → 国家法律法规库需CamoFox，或手动下载后存数据库
4. **裁判文书** → 放弃爬取，考虑淘宝式API或人工录入
