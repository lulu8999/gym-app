# BLE 智能门锁三层分级分析方法

从本项目实战研究中总结的三层分级法，指导每种锁该用什么策略。

## 三层分级

高端锁（2000+ 元） — 米家全自动 / 德施曼 / Schlage / August
  安全芯片双向认证（SE + Secure Boot）
  策略：监控路线（只读不写）或 物理方案
  BLE 破解不可行

中端锁（800-2000 元） — TTLock / 涂鸦Tuya方案锁
  有加密但密钥可从配套 App 提取
  策略：先配对提取 eKey → ESP32 续用
  BLE 破解可行（需先拥有一把）

廉价锁（300-800 元） — 淘宝公模/杂牌 BLE 密码锁
  明文 PIN / 简单 AES 加密
  策略：ESP32 直连写 GATT Characteristic
  ESP32 站门口就能开

## 各层级策略

### 廉价锁 — BLE 直连
- ESP32 扫描 BLE 设备 → 匹配 Service UUID
- 连上后读/写 Characteristic
- 密码通常是明文或 XOR/AES-ECB（密钥硬编码）

### 中端锁 — 配对提取
- 先正常用手机 App 配对一次
- 从越狱/root过的手机或抓包提取 eKey
- 将 eKey 写入 ESP32 固件
- 局限：需要先有一把锁来练手

### 高端锁 — 不可 BLE 破解
- 安全芯片（SE050 / 国产替代）存储密钥
- 私钥物理不可读（安全芯片防拆/防侧信道）
- 替代方案：监控路线或物理方案（舵机/继电器直驱）

## 为什么高端锁没有公开逆向方法

三层次解释（用户偏好）：

1. 技术层：安全芯片物理防读 + firmware 加密签名 + Secure Boot + 双向认证防重放

2. 法律层：DMCA 第 1201 条反规避条款禁止发布破解方法

3. 经济层：逆向一把 $300 的锁需 $30000+ 的设备 + 数周时间，OTA 补丁一次废掉

## 推荐项目路径

第一步：买 TTLock 或涂鸦中端锁练手 → BLE 抓包 → 提取 eKey → ESP32 遥控开

第二步：构建全品牌数据库 → 搜集协议/加密方式/安全等级

第三步：高端锁部署监控方案 → ESP32 监听广播 → 记录开门事件/电量告警
