# 第4课：rm / rmdir（删除文件和目录）

## 核心命令

```bash
rm 文件名              # 删除文件
rm -r 文件夹           # 递归删除（删除文件夹及内部所有内容）
rm -rf 文件夹          # 强制递归删除（不询问）
rmdir 空文件夹         # 只能删除空目录
```

## 新手常见问题

1. **rm 和 rmdir 的区别**
   - `rm` 只能删文件
   - `rmdir` 只能删空目录
   - 删非空目录必须用 `rm -r`

2. **危险命令**
   - `rm -rf /` = 删光整个系统（不可逆！）
   - `rm -rf *` = 删光当前目录所有文件
   - 删除前用 `ls` 确认目录

3. **删除提示**
   - Mac/Linux 默认不提示，直接删
   - 可以用 `rm -i` 逐个确认（但这位用户不需要，太麻烦）
