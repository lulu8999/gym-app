# 第11课：vim 文本编辑器

**日期：** 2026-06-13

## 核心内容

vim 是终端里的文本编辑器，服务器上没有图形界面时改配置文件只能靠它。

### 两种模式
- **普通模式**（默认）→ 看文件、移动光标
- **编辑模式** → 打字修改内容

### 模式切换
```
普通模式 → 按 i → 编辑模式（左下角出现 -- INSERT --）
编辑模式 → 按 Esc → 普通模式
```

### 常用操作
| 按键 | 作用 |
|------|------|
| `i` | 进入编辑模式 |
| `dd` | 删除一整行 |
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 不保存强制退出 |
| `/关键词` | 搜索关键词（按 n 下一个） |

### 口诀
> 进去按 `i`，改完按 `Esc`，保存退出 `:wq`

## 常见问题
- 打开 vim 后敲键盘没反应 → 没进编辑模式，先按 `i`
- `dd` 不需要加冒号，直接在普通模式按两下 d
- `:wq`、`:q!` 的冒号是半角冒号，不是中文冒号
