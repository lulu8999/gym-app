# 第4课：cp / mv（复制和移动）

## 核心命令

```bash
# 复制文件
cp 源文件 目标文件
cp -r 源文件夹 目标文件夹    # 复制整个文件夹

# 移动/重命名
mv 源文件 目标文件
mv 源文件夹 目标文件夹
```

## 常见错误

1. **复制目录**：必须加 `-r`（递归）
   - ❌ `cp dir1 dir2` → 报错
   - ✅ `cp -r dir1 dir2` → 成功

2. **覆盖文件**：默认直接覆盖，不提示
   - 小心同名文件被覆盖

3. **相对路径 vs 绝对路径**
   ```bash
   # 相对路径（从当前位置开始）
   cp file.txt ../backup/
   
   # 绝对路径（从根目录开始）
   cp file.txt /home/lulu/backup/
   ```
