## 能力缺口（按优先级）

| 能力 | 优先级 | 现状 |
|------|--------|------|
| 网页浏览 | P0 | OpenClaw browser-automation 已装但需解决授权 |
| 语音消息处理 | P1 | 目前无需求 |
| 文档深度解析 | P1 | 工具就绪未实战 |
| 持久知识库 | P2 | 记忆+技能够用 |
| 跨 Agent 任务看板 | P2 | 无 |

## 实施路线

### Phase 1：补齐基础
- 解决 OpenClaw 网关/scope 授权 → 浏览器可用
- 或直接装 Playwright 到 Hermes
- 测试 OpenClaw CLI 实际任务执行

### Phase 2：多 Agent 编排
- 分发标准自动化
- 任务看板
- 进度自动汇报

### Phase 3：深化
- Agent 间消息传递（Claude Code 写完→自动审查）
- 共享知识库
