# 21st.dev 设计风格参考系统

> 来源：https://21st.dev/community/components
> 素材路径：/root/frontend-assets/21st-dev/
> 技术栈：React + Tailwind CSS + Next.js

---

## 设计趋势总结（2026）

1. **深色主题**是主流，科技感强，适合 B 端/SaaS
2. **渐变模糊**效果很受欢迎，营造氛围感
3. **大字体+留白**的极简风格
4. **圆角+阴影**是标配，卡片悬浮感
5. **黑白灰中性色**为主，点缀色仅用于 CTA
6. **3D等距图形**作为品牌标识
7. **仪表盘网格布局**展示复杂信息
8. **故障/残影动效**用于文字效果

## 组件分类与风格指南

### 1. Heroes（首屏大图）- 73个组件

**6种主流风格：**

| 风格 | 配色 | 布局 | 适合场景 |
|------|------|------|----------|
| 暗黑3D科技风 | 纯黑+银灰线框 | 图形主导，文字辅助 | AI/云计算/元宇宙 |
| 模糊氛围感 | 深黑+青紫渐变 | 面化沉浸式 | 音乐/艺术/潮流品牌 |
| 复古图文风 | 白底+棕褐照片 | 非对称自由布局 | 个人主页/创意工作室 |
| 深色产品风 | 深紫渐变+大字体 | 居中聚焦 | SaaS产品/软件 |
| 极简CTA风 | 白底+双按钮 | 居中功能布局 | 应用推广/简洁风格 |
| 网格图案风 | 浅灰+细网格 | 居中功能布局 | 设计系统/工具类 |

**设计要点：**
- Hero 是用户第一眼看到的内容，必须快速传递核心价值
- 深色 Hero 适合科技/专业感，浅色 Hero 适合友好/商务感
- CTA 按钮要醒目，放在视觉焦点位置
- 3D 元素能提升高级感，但要控制加载性能

---

### 2. Cards（卡片）- 79个组件

**6种主流风格：**

| 风格 | 特点 | 适合场景 |
|------|------|----------|
| 视觉图像卡 | 彩色图像+文字 | 作品集/品牌宣传 |
| 定价对比卡 | 双栏套餐对比 | SaaS定价页 |
| 纯文字内容卡 | 左对齐文字排版 | 博客/资讯摘要 |
| 案例轮播卡 | 水平图片轮播 | 客户案例展示 |
| 表单型卡 | 输入框+按钮 | 登录/注册 |
| 深色活动卡 | 深底+白文字 | 高端活动/会员 |

**设计要点：**
- 圆角矩形+轻微投影是标配，营造悬浮感
- 信息层级通过字体大小、字重、间距区分
- 悬停反馈：阴影加深、轻微上移
- 卡片内部结构统一：标题+内容+操作区

---

### 3. Buttons（按钮）- 130个组件

**6种主流风格：**

| 风格 | 配色 | 形状 | 交互 |
|------|------|------|------|
| 拟物圆形按钮 | 蓝/红纯色+内发光 | 圆形 | 按压下沉+光晕 |
| 输入框配套按钮 | 黑底白字 | 方形小矩形 | 与输入框联动 |
| 光泽质感按钮 | 渐变光泽 | 圆角矩形 | 光泽扫过动效 |
| 扁平文字按钮 | 黑底+箭头 | 圆角矩形 | 阴影变化 |
| 带边框轻按钮 | 白底+深边框 | 圆角矩形 | 背景色变化 |
| 极简图标按钮 | 纯黑纯白 | 小圆形 | 微缩放 |

**设计要点：**
- 按钮视觉强度要与功能优先级匹配
- CTA 按钮用高对比色，次要按钮用低对比
- 悬停反馈要明确：颜色变化、阴影、位移
- 状态切换：默认→悬停→点击→禁用

---

### 4. Features（功能展示）- 36个组件

**6种主流风格：**

| 风格 | 特点 | 适合场景 |
|------|------|----------|
| 组件集合展示 | Shadcn+Tailwind标签 | 产品推广 |
| 深色仪表盘 | 2×2网格卡片 | SaaS后台 |
| 项目展示轮播 | 3列子卡片+箭头 | 案例展示 |
| 功能清单列表 | 勾选图标+文字 | 产品特性说明 |
| 案例图片轮播 | 水平滚动 | 作品集 |
| 大字体品牌展示 | 粗体+星星点缀 | 品牌标题 |

**设计要点：**
- 功能展示要突出核心价值，不要堆砌
- 用图标+文字组合，比纯文字更易读
- 轮播/水平滚动是案例展示的标准交互
- 3D等距图形作为品牌标识很有效

---

### 5. Navigation Menus（导航菜单）- 16个组件

**9种主流风格：**

| 风格 | 布局 | 适合场景 |
|------|------|----------|
| 后台式垂直导航 | 图标+文字垂直排布 | 企业后台/SaaS |
| 胶囊式水平导航 | 居中圆角按钮 | 轻量化页面 |
| 极简文字导航 | 仅文字无边框 | 极简品牌站 |
| 图标辅助导航 | 每项配小图标 | 工具类产品 |
| 分组导航 | 深色标题+二级导航 | 知识库/内容平台 |
| 复合导航 | 侧边+顶部组合 | 复杂平台 |
| 创意网格导航 | 网格布局文字 | 艺术/设计站 |

**设计要点：**
- 导航是页面的"路标"，不需要抢夺视觉焦点
- 中性色（黑白灰）是通用选择，适配所有品牌
- 激活状态要明确：背景填充、文字加深、下划线
- 多层级导航支持展开/折叠

---

### 6. Texts（文字效果）- 58个组件

**4种主流风格：**

| 风格 | 效果 | 适合场景 |
|------|------|----------|
| 故障艺术 | 抖动、色块位移、条纹撕裂 | 科技/游戏/潮流 |
| 模糊涂抹 | 动态模糊、消失出现 | 创意机构/艺术 |
| 残影运动 | 重复元素+透明度变化 | 运动品牌/汽车 |
| 图形化破坏 | 裁剪、位移、叠加 | 品牌标志/海报 |

**设计要点：**
- 文字效果要服务于内容，不是炫技
- 深色背景能最大化突出发光效果
- 动效要轻量，保证流畅性
- 设计为悬停/加载/激活等状态触发

---

### 7. Forms（表单）- 23个组件

**3种主流风格：**

| 风格 | 配色 | 适合场景 |
|------|------|----------|
| 浅色标准表单 | 白底+黑文字 | 日常办公/信息展示 |
| 深色模式表单 | 黑底+白文字 | 夜间模式/设计工具 |
| 彩色CTA表单 | 白底+蓝色按钮 | 需要强引导的场景 |

**设计要点：**
- 单列/双列极简结构，留白充足
- 输入框、标签、按钮间距统一
- 错误状态用红色边框+提示文字
- 按钮要放在视觉引导线的终点

---

### 8. Inputs（输入框）- 102个组件

**设计要点：**
- 白色填充+浅灰色边框是标配
- 聚焦时边框颜色加深
- 支持状态反馈：错误、字符限制、必填标识
- 带右侧操作按钮的输入框适合"输入+快捷操作"
- 占位符文字用浅灰色，不抢焦点

---

### 9. Tabs（标签页）- 38个组件

**3种主流风格：**

| 风格 | 选中状态 | 适合场景 |
|------|----------|----------|
| 背景对比款 | 黑/深灰背景 | 功能优先场景 |
| 下划线款 | 细/粗下划线 | 内容优先场景 |
| 标注款 | 带"Selected"标签 | 需要引导的场景 |

**设计要点：**
- 「顶部标签栏+下方内容区」是经典布局
- 选中状态要有明确视觉区分
- 切换动效要轻量：背景渐变、下划线滑动
- 标签尺寸统一，仅通过装饰区分样式

---

### 10. Dialogs/Modals（弹窗）- 37个组件

**设计要点：**
- 深色弹窗在浅色界面中形成强层级对比
- 标准结构：顶部标题+中间内容+底部操作
- 按钮用高饱和亮色（蓝/绿）作为视觉锚点
- 支持多种交互：输入、选择、确认、状态反馈
- 弹出/关闭动效要流畅

---

### 11. Selects（选择器）- 62个组件

**设计要点：**
- 白色背景+浅灰细边框是标配
- 支持多状态：收起、展开、禁用、必填
- 深色模式用蓝/灰高亮选中项
- 带输入的选择器适合选项量大的场景
- 原生风格适合性能敏感场景，自定义风格适合品牌统一

---

### 12. Tables（表格）- 30个组件

**2种主流风格：**

| 风格 | 配色 | 适合场景 |
|------|------|----------|
| 亮色表格 | 白底+黑文字 | 日常办公/内容管理 |
| 暗色表格 | 黑底+白文字 | 开发工具/运维监控 |

**设计要点：**
- 「顶部功能区+数据表格区」是基础布局
- 支持筛选、排序、分页、行操作
- 行高适中，兼顾紧凑性与可读性
- 响应式适配：列自适应或横向滚动

## 配色速查表

### 基础色板
- **背景**：#ffffff（白）、#f7f7f7（浅灰）、#000000（黑）
- **文字**：#111111（深灰）、#666666（中灰）、#999999（浅灰）
- **边框**：#e5e5e5（浅灰）、#333333（深灰）

### 强调色
- **主CTA**：#000000（黑底白字）、#2563eb（蓝色）
- **成功**：#22c55e（绿色）
- **错误**：#ef4444（红色）
- **警告**：#f59e0b（橙色）

### 深色模式
- **背景**：#0a0a0a、#111111、#1a1a1a
- **文字**：#ffffff、#a1a1aa、#71717a
- **边框**：#27272a、#3f3f46

## 布局模式速查

### 页面级布局
1. **侧边导航+内容区**：后台/SaaS（最常见）
2. **顶部导航+内容区**：品牌站/营销页
3. **居中单列**：落地页/博客
4. **全屏沉浸**：作品集/艺术站

### 组件级布局
1. **网格 Grid**：卡片、图片展示
2. **列表 List**：文章、数据
3. **轮播 Carousel**：案例、推荐
4. **瀑布流 Masonry**：作品、图片

---

## 交互模式速查

### 基础反馈
- 悬停：阴影加深、轻微上移、颜色变化
- 点击：按下下沉、颜色反转
- 聚焦：边框高亮、外发光

### 状态变化
- 加载：骨架屏、进度条、旋转图标
- 成功：绿色对勾、toast 提示
- 错误：红色边框、错误文字

### 动效原则
- 轻量优先：200-300ms 过渡
- 有意义的动效：引导注意力，不是炫技
- 性能优先：CSS > JS 动画

## 建站决策树

### 选择 Hero 风格
```
科技/AI/工具 → 暗黑3D科技风
SaaS/企业 → 深色产品风 or 极简CTA风
艺术/创意 → 模糊氛围感 or 复古图文风
品牌/营销 → 网格图案风 or 大字体品牌展示
```

### 选择配色方案
```
B端/SaaS → 黑白灰+蓝色CTA
品牌/创意 → 黑白灰+品牌色CTA
消费级 → 浅色为主+鲜艳CTA
暗黑主题 → 深灰背景+白文字+蓝色CTA
```

### 选择导航类型
```
功能多/层级深 → 垂直导航
内容优先 → 极简水平导航
品牌/创意 → 创意网格导航
轻量工具 → 胶囊式导航
```
