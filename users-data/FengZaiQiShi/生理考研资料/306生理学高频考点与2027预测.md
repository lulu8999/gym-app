#### 3. 突触传递（第10章）
- **EPSP vs IPSP**（兴奋性vs抑制性突触后电位）

- **递质释放过程**（Ca²⁺内流、囊泡融合）

#### 4. 感觉传入（第10章）
- **特异性vs非特异性投射系统**（区别表格必背）
- **痛觉**（快痛慢痛传导通路）
- **牵张反射**（腱反射vs肌紧张）

### 🟠 第二梯队（高频选择）

- **G蛋白偶联受体**（cAMP-PKA、IP3-DAG通路）

#### 6. 跨膜转运（第2章）
- **原发性vs继发性主动转运**（Na⁺-K⁺泵、Na⁺-葡萄糖同向转运）

#### 7. 激素调节（第11章）
- **甲状腺激素**（合成、调节、生理作用）
- **胰岛素vs胰高血糖素**（血糖调节）

#### 8. 呼吸调节（第5章）
- **化学感受器**（中枢vs外周，CO₂/H⁺/低O₂作用）

## 📝 各章节核心考点速记

### 第1章 绪论（1分，了解）
- 内环境 = 细胞外液
- 稳态 = 动态平衡
- 神经调节（快而准）vs体液调节（慢而久）
- 正反馈（排尿、分娩、凝血）vs负反馈（血压、体温）

### 第2章 细胞功能（6分，重点）
**跨膜转运：**
- 单纯扩散：O₂、CO₂、乙醇、类固醇
- 易化扩散：经通道（离子）+ 经载体（葡萄糖入红细胞）
- 主动转运：原发（Na⁺泵）+ 继发（葡萄糖入肠上皮）
- 入胞出胞：大分子物质

**信号转导：**
- 离子通道型（烟碱型Ach受体）
- G蛋白偶联（M型Ach受体、肾上腺素受体）
- 酶联型（酪氨酸激酶受体：胰岛素、生长因子）

**电生理：**
- 静息电位 = K⁺平衡电位（外正内负）
- 动作电位 = Na⁺内流（去极化）+ K⁺外流（复极化）
- 阈电位：能触发动作电位的临界膜电位
- 局部电位：等级性、可总和、不衰减（与动作电位区别）

### 第3章 血液（2分，了解）
- 血细胞比容：男40-50%，女37-48%
- 血浆蛋白：白蛋白（维持胶渗压）、球蛋白（免疫）、纤维蛋白原（凝血）
- 红细胞生成：EPO（肾产生）、Fe²⁺、VitB12、叶酸
- 凝血因子：内源性（Ⅻ→Ⅺ→Ⅸ→Ⅷ）vs外源性（Ⅲ→Ⅶ）
- 血型：ABO（红细胞抗原）+ Rh（D抗原）

### 第4章 血液循环（8分，重中之重）
**心肌电生理：**
- 心室肌：快反应细胞（0期Na⁺、2期Ca²⁺-K⁺平台期）
- 窦房结：慢反应细胞（0期Ca²⁺、4期自动去极化）
- 传导性：房室延搁（防房室收缩重叠）
- 有效不应期：心室肌200-300ms（长于收缩期）

**泵血功能：**
- 心动周期：房缩期→等容收缩期→射血期→等容舒张期→充盈期
- 每搏输出量（SV）×心率（HR）=心输出量（CO）
- 影响心输出量因素：前负荷（Starling机制）、后负荷（动脉压）、心肌收缩力

**血管生理：**
- 动脉血压形成：心输出量+外周阻力
- 收缩压反映搏出量，舒张压反映外周阻力
- 脉搏压=收缩压-舒张压（老年人大）
- 中心静脉压：右心房压，反映回心血量

**心血管调节：**
- 压力感受器反射（窦弓反射）：减压反射（血压升高→心率减慢）
- 肾上腺素：强心（β₁）
- 去甲肾上腺素：升压（α₁收缩血管）

### 第5章 呼吸（5分，重点）
- 肺通气：肺泡与外界气体交换（动力：呼吸肌运动）
- 肺换气：肺泡与血液间O₂/CO₂交换（动力：分压差）
- 氧解离曲线：S型，右移（酸/热/2,3-DPG增加→释放O₂增加）
- 化学感受器：中枢（对CO₂敏感，通过H⁺作用）+ 外周（颈动脉体，对低O₂敏感）
- CO₂是调节呼吸的最重要生理性因素

### 第6章 消化（3分，一般）
- 唾液：淀粉酶（初步消化淀粉）
- 胃液：盐酸（激活胃蛋白酶原、杀菌）、胃蛋白酶（分解蛋白质）、内因子（促进VitB12吸收）
- 胰液：最强消化液（碳酸氢盐+多种酶）
- 胆汁：乳化脂肪（无酶）
- 胃肠激素：胃泌素、促胰液素、缩胆囊素

### 第7章 能量代谢与体温（2分，了解）
- 基础代谢率：清醒、安静、空腹状态下能量代谢
- 体温调定点：视前区-下丘脑前部（PO/AH）
- 散热方式：辐射（主要）、传导、对流、蒸发（高温时唯一方式）

### 第8章 泌尿（4分，重点）
- 肾小球滤过：有效滤过压=毛细血管压-(血浆胶渗压+囊内压)
- 肾糖阈：尿中开始出现葡萄糖的血糖浓度（180mg/100ml）
- 抗利尿激素（ADH）：下丘脑合成，垂体释放，促进水重吸收
- 醛固酮：保钠排钾保水

### 第9章 感觉器官（2分，了解）
- 眼折光：角膜、房水、晶状体、玻璃体
- 视近调节：晶状体变凸（最重要）、瞳孔缩小、眼球汇聚
- 视力：视角1分
- 视野：鼻侧小、颞侧大；白色最大、绿色最小

### 第10章 神经系统（7分，重中之重）
**突触传递：**
- 经典突触：轴突-树突/胞体
- EPSP：Na⁺内流，去极化，兴奋性
- IPSP：Cl⁻内流，超极化，抑制性
- 递质：Ach（胆碱能）、NE（肾上腺素能）、多巴胺、5-HT、GABA、谷氨酸

**感觉传入：**
- 特异投射系统：点对点投射到皮层特定区域（产生特定感觉）
- 非特异投射系统：弥散投射（维持觉醒）
- 快痛：Aδ纤维，尖锐定位清楚
- 慢痛：C纤维，钝痛定位模糊

**运动调节：**
- 牵张反射：腱反射（位相性）+ 肌紧张（紧张性）
- 去大脑僵直：伸肌紧张性亢进（中脑上、下丘间切断）
- 基底神经节：帕金森（多巴胺少，肌紧张过高）+ 舞蹈病（肌紧张过低）
- 小脑：维持平衡、调节肌紧张、协调随意运动

**自主神经：**
- 交感：应激（心跳快、出汗、瞳孔大、支气管扩张、血糖高）
- 副交感：安静（心跳慢、消化加强、瞳孔小）

### 第11章 内分泌（6分，重点）
**下丘脑-垂体：**
- 下丘脑调节肽：TRH、GnRH、GHRH、CRH、GHIH
- 腺垂体：生长激素（GH）、TSH、ACTH、FSH/LH、PRL
- 神经垂体：ADH、催产素（下丘脑合成）

**甲状腺：**
- T3、T4：提高基础代谢率、促进生长发育（对脑和骨最重要）
- 呆小症：幼年缺乏（智力低+身材矮）
- 单纯性甲状腺肿：缺碘（代偿性肿大）

**肾上腺：**
- 糖皮质激素：升高血糖、抗炎、抗过敏、应激
- 盐皮质激素（醛固酮）：保钠排钾
- 儿茶酚胺：肾上腺素、去甲肾上腺素

**胰岛：**
- 胰岛素：降血糖（促进糖利用、糖原合成、脂肪合成、蛋白质合成）
- 胰高血糖素：升血糖

### 第12章 生殖（2分，了解）
- 睾酮：精子发生、副性征、性欲
- 雌激素：促进卵泡发育、子宫增生、副性征
- 孕激素：维持妊娠、子宫分泌期、产热
- 月经周期：卵泡期（雌激素）→ 排卵（LH峰）→ 黄体期（孕激素）

### 病例一：心力衰竭与水钠代谢紊乱

**【病史】**
男性，68岁。因"活动后气促、下肢浮肿2周"入院。患者有高血压病史约20年，服用氨氯地平缓释片控制血压。近2周来出现活动耐力明显下降，爬一层楼梯即感气促，伴夜间阻、双下肢水肿。查体：血压165/100mmHg，心率102次/分，双下肢凹陷性水肿，颈静脉怒张。心电图：左室肥厚伴ST-T改变。

**【问题】**
1. 该患者下肢水肿的主要发生机制是什么？请从Starling力平衡角度分析。
2. 患者出现心率加快的代偿机制及其生理意义是什么？
3. 分析该患者心输出量降低的可能因素（前负荷、后负荷、心肌收缩力）。
4. 分析氨氯地平治疗该患者的作用机制及注意事项。

**【参考答案要点】**
1. 心力衰竭→心输出量↓→肾流量↓→RAAS激活→钠水保留。Starling力分析：出球小动脉端滤过压升高+血浆胶体渗透压因钠水潴留上升→组织液生成↑。
2. 心率加快是交感神经兴奋对低心输出量的代偿，通过Frank-Starling机制增加心输出量，但本患者心功衰竭严重，代偿有限。
3. 前负荷增加（回心血量残留）；后负荷增加（动脉压）；心肌收缩力下降（高血压性心脏病变）。
4. 阻断Ca²⁺通道→平滑肌舒张→外周阻力↓→后负荷↓。注意：防止体位性低血压。

### 病例二：糖尿病酮症酸中毒与气代谢紊乱

**【病史】**
女性，32岁。因"多饮、多尿、体重下降三个月，恶心、呕吐、深大呼吸3天"入院。体检：BP 90/60mmHg，脉细速，皮肤发凉，呼吸深大35次/分，呼出气有烂苹果味。血糖：26.8mmol/L；血钠：132mmol/L；血气：pH 7.18，PaCO₂ 18mmHg，HCO₃⁻ 8mmol/L；尿酮体：(✓)。

**【问题】**
1. 该患者发生代谢性酸中毒的主要发生机制是什么？
2. 请解释该患者呼吸深大的发生机制及其生理意义。
3. 分析该患者水电解质紊乱的特点及其发生机制。
4. 从抗利尿激素、肾上腺素出发，分析该患者的液体管理原则。

**【参考答案要点】**
1. 胰岛素绝对或相对不足→肌糖不能利用→脂肪分解↑→酮体生成↑→代谢性酸中毒。
2. 代偿性Kussmaul呼吸：[H⁺]↑→外周化学感受器兴奋→呼吸中枢兴奋→呼吸加深加快→CO₂排出增多→尽可能缓冲酸中毒。
3. 高血糖→渗透性利尿→多尿→脱水；酸中毒→K⁺外流（与H⁺交换）→高钾血症。
4. 抗利尿激素缺乏→水重吸收↓；低容量性低钠伴高钾时，先补液后补钠；确定尿量后用肾上腺素。

### 病例三：高血压急症与神经内分泌调节

**【病史】**
男性，52岁。因"头痛、视物模糊、胸闷半天"急诊。测血压220/140mmHg，血压急升时伴有面色苍白、出冷汗、心率120次/分。给予硝普钠后血压逐渐下降，但出现头痛、心悸、面色潮红。

**【问题】**
1. 分析该患者面色苍白出冷汗的交感神经反射机制。
2. 硝普钠过度降压后出现头痛面色潮红，请用压力感受器反射解释其发生机制。
3. 分析急性升压期间肾上腺素、儿茶酚胺的分泌变化及其作用。
4. 对于此患者，降压过程中为什么要避免血压过快下降？

**【参考答案要点】**
1. 剧烈升压→窦弓反射强烈兴奋→交感神经紧张性↑→皮肤血管收缩、汗腺分泌↑。
2. 硝普钠→血管扩张→压力感受器刺激↓→反射性交感抑制↓→副交感兴奋→血管扩张、心率↓过度。
3. 急性应激：肾上腺素、儿茶酚胺大量释放→升高心输出量、外周阻力，维持脑冠灌注。
4. 避免脑冠灌注压过快降低→脑缺血；避免压力感受器反射性调节紊乱。

### 病例四：帕金森病与基底神经节功能紊乱

**【病史】**
男性，65岁。因"双上肢震颤、行动迟缓逐渐加重2年"就诊。查体：面具脸，双上肢肌张力增高呈铅管样，智力正常。加强试验出现静止性震颤，动作迟缓。给予左旋多巴治疗后症状明星改善。

**【问题】**
1. 请解释该患者肌张力增高的神经生理机制。
2. 从突触传递角度，分析多巴胺通路与该病发病的关系。
3. 该患者的反射性可有何变化？为什么？
4. 分析黑质-纹状体通路与该患者典型表现的关系。

**【参考答案要点】**
1. 黑质-纹状体通路多巴胺能神经元变性→纹状体多巴胺能神经元抑制↓→大脑皮层对延髓的抑制↓→延髓网状构出取力↑→肌张力增高（铅管样强直）。
2. 黑质内多巴胺能神经元退变→尾状核多巴胺能神经元功能相对亢进→肌张力进一步增高。
3. 肌紧张反射亢进：肌棱内肌筋受器敏感性增高，肌紧张反射增强。
4. 黑质抑制缓解，出现相反表现：抖动（静止性）和运动减少（出现典型弥张步态）。

### 病例五：急性尿潴留与抗利尿激素分泌调节

**【病史】**
男性，58岁。因"小便排尿困难、腹部病性包块1天"入院。患者有良性前列腺增生史。查体：腹部积增按之浓淋，扬展存在于肛门以上，尿畜留。诊断：急性尿潴留。导尿后小便恢复流畅，量约2000ml。导尿后第2天出现多尿（日尿量4500ml）。

**【问题】**
1. 分析尿潴留导致尿滤的发生机制。
2. 该患者导尿后出现多尿的发生机制是什么？请分析其液体量变化特点。
3. 从肾小球滤过和重吸收角度，分析尿潴留时为什么容易发生氮中毒。
4. 分析该患者抗利尿激素（ADH）分泌变化及其调节机制。

**【参考答案要点】**
1. 尿潴留→肾盂内压↑→有效滤过压↓→趋零→尿生成停止。
2. 粘性渗压性利尿→渗透性利尿期多尿。机制：尿潴留时渗透性利尿↑→水重吸收↓；压力性利尿建立后渗透性利尿回到正常→原有水分重新吸收→多尿。
3. 肾小球滤过从内进行→尿素等运出受阻→血尿素氮↑→氮中毒。
4. 尿潴留时：渐进性膀胱扩张→盂内压感受器受刺激→盂肩反射抑制（盂内压感受器）→ADH↓→水重吸收↓。

### 病例六：II型呼衰与肺泡通气功能障碍

**【病史】**
男性，72岁。COPD病史约30年，因"咳嗽、咳痰加重、气急3天"入院。查体：表情痛苦，口唇发红，球结膜充血，周围淋巴水肿。呼吸频率32次/分，节律不整。血气：pH 7.28，PaO₂ 48mmHg，PaCO₂ 78mmHg，HCO₃⁻ 36mmol/L。

**【问题】**
1. 该患者呼吸衰竭的主要类型及其发生机制。
2. 分析患者出现呼吸节律不整的可能原因。
3. 从化学感受器角度解释为什么高碳酸血症会出现忽视低氧血症。
4. 分析此患者写如何进行氧疗不当会导致二氧化碳排出妥恶化。

**【参考答案要点】**
1. II型呼衰（泛化性缺氧）。机制：COPD→残气量↑→肺泡通气功能降低→低氧和CO₂潴留。
2. 肺性脑病（CO₂潴留→H⁺↑→脑细胞酸中毒）或低氧血症。
3. CO₂冗还性：长期CO₂潴留→中枢化学感受器适应（对CO₂敏感性↓）→依赖外周低氧刺激维持呼吸。
4. 高浓度氧疗→外周低氧被抵消→呼吸驱动↓→CO₂排出更少→脏酸中毒恶化。

### 病例七：体位性低血压与自主神经反射

**【病史】**
女性，24岁。因"久站时头晕、视力模糊、大汗底下的人可以休息一下"就诊。患者无体位性低血压三年，体检当日未病。午餐后起立站立时突发眼前发黑、心慌、出冷汗，但亏才未怪。即刻躺下后症状缓解。测平卧位血压110/70mmHg，站立3分钟后80/50mmHg。

**【问题】**
1. 请分析该患者站立时出现眼前发黑、出冷汗的发生机制。
2. 从交感神经与心血管反射角度，解释该患者体位性低血压的发生机制。
3. 分析该患者危险导尿的反常反应及其发生机制。
4. 为什么该患者危险导尿后真性延迟、回心血量减少？

**【参考答案要点】**
1. 站立→下肢静脉储血↑→回心血量↓→心输出量↓→脑冠灌注↓→视觉、交感神经控制的汗腺功能抗议。
2. 低容量调节失败：站立→回心血量↓→心输出量↓，但心率增快不足以代偿；压力感受器反射出现时间延迟。
3. 压力性利尿→延迟出现的低血压反射（下肢静脉张受器受拔军艰彩）。
4. 危险导尿→脉率性尿性回流↑→真性延迟、回心血量↓→低血压恶化。

### 病例八：心肌检塞与均衡调节

**【病史】**
男性，48岁。因"发作性胸痛两天，加重伴恶心呕吐半天"入院。患者向有高血压、高血脂史。诊断：急性广泛前壁心肌检塞。予以以下战略：1、血栓核力融；2、合并心美不全行主动脉球囊成型。血压从入院时110/70mmHg逐渐降至85/55mmHg，心率从砂况发生89次/分逐渐升至118次/分。

**【问题】**
1. 分析该患者血压下降时心率加快的发生机制及其生理意义。
2. 用Starling机制解释为什么心美不全时心输出量会显著下降。
3. 分析压力感受器反射在该患者生理调节中的作用及其限制性。
4. 从代偿机制角度，解释为什么急性心肌检塞时心率加快有一定保护作用。

**【参考答案要点】**
1. 心输出量↓→剧雇压→窦弓反射兴奋→心率↑。意义：维持心输出量、保证脑冠灌注。
2. 心肌抓尿力↓→剩余血量↑→舟张末压↑（前负荷）→心输出量↓（心功曲线下移）。
3. 心输出量↓→血压↓→压力感受器刺激↓→反射性交感抑制↓→心率↑、外周阻力↓、心肌收缩力↑。限制：过度依赖会增加心肌氧耗。
4. 代偿性：通过Frank-Starling机制增加每搏输出量；代价：心率快→冠脉灌注时间↓→心肌氧供↓。

## 病例分析题复习建议

### 答题技巧
1. **先读题，再看病史**：明确问题侧重哪章节
2. **拉知识框架**：从角色、作用、与其他过程的联系三个维度分析
3. **不要漏层次**：尽可能从细胞→器官→系统→整体调节多层次分析
4. **关键词标记**：在病史中圈出与问题相关的临床表现

### 典型错误
1. **忘记代偿机制**：仅描述发生了什么，未解释为什么
2. **混淆概念**：如将"前负荷"与"后负荷"混淆
3. **答非所问**：问机制却只写表现
4. **遗漏关键词**：如"对于该患者的意义"等语句
